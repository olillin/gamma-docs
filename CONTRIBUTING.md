# Contributing

Thank you for your interest in contributing to the Gamma documentation!

Contributions to this document are welcome and much appreciated. Contact Cal or
digIT to find out how you can help.

## Formatting

This project uses [mdformat](https://mdformat.rtfd.io) and
[Alejandra](https://github.com/kamadorueda/alejandra) to keep all files looking
nice and consistent.

If using Nix you can start the development shell including both of these using:

```console
nix develop
```

Run this command to format Markdown files with mdformat:

```console
mdformat .
```

Run this command to format Nix files with Alejandra:

```console
alejandra .
```
